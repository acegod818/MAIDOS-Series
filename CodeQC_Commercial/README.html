Quick Use

Expand-Archive .\releases\CodeQC_Commercial_Windows_x64_0.4.0.zip -DestinationPath .\local-codeqc-commercial
.\local-codeqc-commercial\codeqc-runner.exe version
.\local-codeqc-commercial\codeqc-runner.exe health

Repository Boundary

Allowed here: compiled binary package, checksum, manifest, and usage or policy documents.

Forbidden here: commercial source trees, Python source packages, wheels, sdists, build specs, virtual environments, caches, and runtime databases.
