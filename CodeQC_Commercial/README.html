CodeQC Commercial Binary Release

This directory is intentionally binary-only.

The commercial CodeQC source code is not committed to this repository. The release artifact is a compiled Windows x64 package:

releases/CodeQC_Commercial_Windows_x64_0.4.0.zip

Package

  • Product: CodeQC Commercial Runner
  • Release ID: gate2-beta-baseline
  • Core version: codeqc-core 0.4.0
  • Runner version: codeqc-runner 0.1.0
  • Platform: Windows x64
  • ZIP SHA256: 2410F1F91F514BAB0F9C7E01A173C9E1D8D1678BAC0FDB0D2F20C3737C05C473

Quick Use

Expand-Archive .\releases\CodeQC_Commercial_Windows_x64_0.4.0.zip -DestinationPath .\local-codeqc-commercial
.\local-codeqc-commercial\codeqc-runner.exe version
.\local-codeqc-commercial\codeqc-runner.exe health
.\local-codeqc-commercial\codeqc-runner.exe selftest --output .\selftest.json
.\local-codeqc-commercial\codeqc-runner.exe report C:\path\to\project --profile advisory --report-dir reports --format html

Repository Boundary

Allowed here: compiled binary package, checksum, manifest, and usage or policy documents.

Forbidden here: commercial source trees, Python source packages, wheels, sdists, build specs, virtual environments, caches, and runtime databases.
