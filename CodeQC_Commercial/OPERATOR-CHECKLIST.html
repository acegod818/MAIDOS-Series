CodeQC Commercial Operator Checklist

Use this checklist before handing the commercial ZIP to a customer, evaluator, or internal stakeholder.

Required Checks

  1. Confirm CodeQC_Commercial/releases/CodeQC_Commercial_Windows_x64_0.4.0.zip exists.
  2. Confirm the ZIP SHA256 matches SHA256SUMS.txt.
  3. Open the latest GitHub Actions run for CodeQC Commercial Binary Gate.
  4. Confirm the job conclusion is success.
  5. Download or inspect the codeqc-commercial-selftest artifact.
  6. Confirm selftest.json reports status: PASS.
  7. Confirm acceptance_matrix.covered_ids contains the selftest-owned acceptance IDs.
  8. Confirm the workflow did not report missing acceptance IDs.

Release Boundary

Do not add commercial source code, wheels, sdists, build specs, virtual environments, caches, or runtime databases to this repository.
