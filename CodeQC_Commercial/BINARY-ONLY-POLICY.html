CodeQC Commercial Binary-Only Policy

CodeQC Commercial is distributed through compiled release artifacts in this repository. Commercial source code must remain outside the public repository.

Allowed

  • Compiled executable packages.
  • Release manifests and checksums.
  • HTML usage guides, license notices, privacy, security, and terms documents.
  • Smoke-test evidence that does not expose commercial source code.

Forbidden

  • Commercial .py, .ts, .rs, .cs, or equivalent source files.
  • Python wheels, source distributions, or package archives that contain source code.
  • Build specifications, virtual environments, caches, temporary reports with local source paths, and runtime databases.
  • Any file that reveals proprietary implementation details rather than product operation.

Release Gate

Before pushing, scan the repository tree and the binary package contents. The release passes only when the committed CodeQC Commercial directory contains no commercial source code.
